# Documentation Assets

This directory contains documentation assets for the Greene County Property Finder.

## Screenshots

Add screenshots of your application here:

- `screenshot.png` - Main application screenshot (for README)
- `screenshot-map.png` - Interactive map view
- `screenshot-analytics.png` - Analytics dashboard
- `screenshot-owner-lookup.png` - Owner lookup page
- `screenshot-data-management.png` - Data management page

## How to Capture Screenshots

1. Run the app: `streamlit run app.py`
2. Navigate to the desired page
3. Use browser developer tools or screenshot tool
4. Recommended size: 1200x800 pixels
5. Save as PNG with descriptive name

## Architecture Diagrams

If you create architecture or flow diagrams, place them here:

- `architecture.png` - System architecture
- `data-flow.png` - Data flow diagram
- `api-diagram.png` - API integration diagram

## Generating Diagrams

You can use tools like:
- [Mermaid](https://mermaid.js.org/) - Text-based diagrams
- [Draw.io](https://draw.io/) - Visual diagram editor
- [Excalidraw](https://excalidraw.com/) - Sketch-style diagrams
